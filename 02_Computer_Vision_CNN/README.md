# 👁️ 02 | Computer Vision & CNNs

Η Τεχνητή Όραση (Computer Vision) είναι ο κλάδος που μαθαίνει στους υπολογιστές να "βλέπουν". Εδώ αφήνουμε τα παραδοσιακά νευρωνικά δίκτυα και περνάμε στα **Συνελικτικά Νευρωνικά Δίκτυα (CNNs)**, τα οποία σαρώνουν τις εικόνες σαν φίλτρα!

### ❓ Ερωτήματα που θα απαντήσουμε:
* Πώς ένας υπολογιστής μετατρέπει τα pixel σε αναγνώριση αντικειμένου;
* Τι είναι το Convolution (Συνέλιξη) και το Pooling (Συμπίεση);
* Πώς χτίζουμε μια αρχιτεκτονική που αναγνωρίζει ψηφία ή ζώα;

### 🛠️ Νέα Εργαλεία:
* **`nn.Conv2d`**: Το φίλτρο που σαρώνει 2D εικόνες ψάχνοντας για ακμές και μοτίβα.
* **`nn.MaxPool2d`**: Το εργαλείο που μικραίνει την εικόνα, κρατώντας μόνο τα σημαντικά pixels για να σώσει μνήμη.
* **Flattening (`torch.flatten`)**: Το "ίσιωμα" ενός πίνακα εικόνας σε μια απλή γραμμή για να βγει το τελικό αποτέλεσμα.
