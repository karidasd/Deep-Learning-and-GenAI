# 🔍 05 | Retrieval-Augmented Generation (RAG)

Το RAG είναι ο λόγος που οι εταιρείες παγκοσμίως επενδύουν δισεκατομμύρια στο AI σήμερα. Επιτρέπει στα 'χαζά' (δημόσια) μοντέλα να αποκτούν ιδιωτική, εξειδικευμένη γνώση, χωρίς να χρειάζονται επανεκπαίδευση (Fine-tuning)!

### ❓ Ερωτήματα που θα απαντήσουμε:
* Τι κάνουμε όταν το ChatGPT δεν ξέρει τα δεδομένα της εταιρείας μας;
* Τι είναι το LangChain και πώς λειτουργεί;
* Πώς 'κόβουμε' ένα μεγάλο έγγραφο σε κομμάτια (Text Splitting) και πώς τα αποθηκεύουμε σε μια Vector Database;

### 🛠️ Νέα Εργαλεία:
* **LangChain**: Το απόλυτο εργαλείο 'διασύνδεσης' των LLMs με εξωτερικά εργαλεία (βάσεις δεδομένων, PDFs, ίντερνετ).
* **Text Splitter**: Εργαλείο που κόβει έξυπνα τα κείμενα (ώστε να μη σπάσει τις λέξεις ή τα νοήματα στη μέση).
* **Vector Database**: (Θεωρητική Εισαγωγή) Μια βάση δεδομένων που αποθηκεύει αριθμούς (Embeddings) αντί για λέξεις.
