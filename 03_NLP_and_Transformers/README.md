# 📝 03 | Natural Language Processing & Transformers

Πώς 'διαβάζουν' οι υπολογιστές ένα βιβλίο; Πώς καταλαβαίνουν αν ένα σχόλιο (review) στο ίντερνετ είναι θετικό ή αρνητικό; Καλώς ήρθατε στο NLP! 

### ❓ Ερωτήματα που θα απαντήσουμε:
* Πώς μετατρέπουμε μια λέξη σε αριθμό (Word Embeddings) έτσι ώστε η λέξη 'Σκύλος' να είναι μαθηματικά κοντά στη λέξη 'Γάτα';
* Τι είναι τα Αναδρομικά Νευρωνικά Δίκτυα (RNN/LSTM) και γιατί χρειάζονται 'μνήμη';
* Ποια είναι η βάση της αρχιτεκτονικής που έφερε την επανάσταση των Transformers;

### 🛠️ Νέα Εργαλεία:
* **`nn.Embedding`**: Το μαγικό επίπεδο που δημιουργεί ένα πολυδιάστατο 'λεξικό'.
* **`nn.LSTM` (Long Short-Term Memory)**: Νευρωνικό δίκτυο που διαβάζει κείμενα λέξη-λέξη, κρατώντας ταυτόχρονα στη μνήμη του τι διάβασε προηγουμένως!
