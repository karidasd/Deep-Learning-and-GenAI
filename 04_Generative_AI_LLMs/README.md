# 🗣️ 04 | Generative AI & Large Language Models (LLMs)

Φτάσαμε στην εποχή του ChatGPT. Το Generative AI είναι ικανό όχι απλώς να αναλύει, αλλά να **δημιουργεί** νέο περιεχόμενο, νέα κείμενα και νέο κώδικα (ακριβώς όπως αυτό που διαβάζετε!).

### ❓ Ερωτήματα που θα απαντήσουμε:
* Τι είναι το Hugging Face και γιατί θεωρείται το "GitHub του AI";
* Πώς μπορούμε να "κατεβάσουμε" ένα γλωσσικό μοντέλο (LLM) στοπ υπολογιστή μας;
* Τι είναι το Prompting στον κώδικα και πώς λειτουργεί το Inference των μοντέλων παραγωγής κειμένου (όπως το GPT-2, LLaMA, κλπ);

### 🛠️ Νέα Εργαλεία:
* **Hugging Face `transformers`**: Η πιο διάσημη open-source βιβλιοθήκη για Large Language Models.
* **`pipeline('text-generation')`**: Το μαγικό εργαλείο που 'κατεβάζει' τα βάρη του μοντέλου και αρχίζει να παράγει κείμενο με 2 γραμμές κώδικα!
